## Common Issues

### If you select wrong option:
- Press **UP/DOWN arrows** to move
- Press **ENTER** to confirm

### If you type wrong text:
- Press **CTRL+A** to select all
- Press **DELETE** to clear
- Type again

### If installation hangs:
- Press **CTRL+C** to cancel
- Wait 5 minutes
- Try again
